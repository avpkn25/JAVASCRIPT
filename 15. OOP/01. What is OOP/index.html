OOP

Object-Oriented Programming (OOP) is a programming language / style organized around objects rather then "actions" and data rather than logic.

JavaScript

Javascript is a versatile and dynamic programming language that supports multiple paradigms, including procedural, functional and object-oriented programming.

JavaScript is not a purely object-oriented language like Java or C++, it provides the tools and features to implement OOP concepts effeciently.

OOP in JS

OOP in JavaScript refers to the practice of designing and structuring code around objects and their interactions.

It is a programming paradigm that allows developers to model real-world entities and their relationships using objects and classes.
