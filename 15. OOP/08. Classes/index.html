Classes

Classes are one of the featurs introduced in the ES6 version of JavaScript

A Class is a blueprint for the object. You can create an object from the class.

You can think of the class as a sketch (prototype) of a house.

It contains all the details about the floors, doors, windows, etc..

Based on these descriptions, you can build the house.

Sine many houses can be made from the same description, we can create many objects from a class.
