Arrow Functions

Arrow functions, also known as fat arrow functions, are a concise and shorter way to define functons in JavaScript.

They were introduced in ECMAScript 6 (ES6) and provide a more compact syntax compared to traditional function expressions

Enhanced Object Literals

Enhanced object literals, introduced in ECMAScript 6 (ES6), are a set of enhancements to the syntax for defining objects in JavaScript.

These enhancements make it more convenient and concise to define object properties and methods

Default function parameters

It was introduced in ECMAScript 6 (ES6), allow you to assign default values to function parameters.

When a function is called, and an argument is not provided for a parameter, the default value will be used instead.

This feature makes it easier to handle cases where arguments might be missing or undefined.
